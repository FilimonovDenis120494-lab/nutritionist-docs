Политика конфиденциальности

Дата публикации: 23 февраля 2026 г. · Редакция 1.0

1. Общие положения

1.1. Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей Telegram-бота «Нутрициолог-бот» (далее — «Бот»).

1.2. Оператором персональных данных является Филимонов Денис Анатольевич, плательщик налога на профессиональный доход (самозанятый) (далее — «Оператор»).

1.3. Используя Бот, Пользователь выражает согласие с настоящей Политикой. Согласие подтверждается нажатием кнопки «Принимаю и начинаем» при первом запуске Бота.

1.4. Обработка персональных данных осуществляется в соответствии с Федеральным законом № 152-ФЗ «О персональных данных» от 27.07.2006 г.

2. Какие данные мы собираем

2.1. Данные, предоставленные Пользователем:

  • Имя
  • Пол
  • Возраст (возрастной диапазон)
  • Рост и вес
  • Уровень физической активности
  • Цели (похудение, набор массы, улучшение здоровья и т.д.)
  • Жалобы на здоровье (из предложенного списка)
  • Хронические заболевания (из предложенного списка)
  • Аллергии и пищевые ограничения

2.2. Данные из загруженных анализов:

  • Числовые показатели лабораторных исследований (маркеры, значения, референсные диапазоны)
  • Фрагменты текста из PDF для корректной интерпретации

Важно: Сам PDF-файл удаляется с сервера сразу после извлечения данных. Хранятся только числовые показатели.

2.3. Технические данные:

  • Telegram ID пользователя
  • Имя пользователя в Telegram (username)
  • Дата и время обращений
  • Статус подписки и история платежей (без данных банковских карт)

2.4. Данные, которые мы НЕ собираем:

  • Фамилию, отчество, паспортные данные
  • Адрес проживания
  • Номер телефона и email
  • Данные банковских карт (обработка платежей — через ЮKassa)

3. Цели обработки данных

3.1. Персональные данные обрабатываются исключительно для:

  • Формирования персонализированных рекомендаций по питанию и нутрициологии
  • Анализа лабораторных показателей и отслеживания динамики
  • Составления индивидуального меню
  • Подбора БАД с учётом индивидуальных особенностей
  • Обработки платежей и управления подписками
  • Отправки напоминаний о повторной сдаче анализов (для платных тарифов)

3.2. Данные не используются для:

  • Рекламных рассылок
  • Продажи третьим лицам
  • Профилирования в маркетинговых целях

4. Передача данных третьим лицам

4.1. Для формирования рекомендаций используется ИИ-сервис Anthropic (США, модель Claude). В ИИ-сервис передаются:

  • Числовые показатели анализов
  • Параметры анкеты (пол, возраст, вес, рост, жалобы, цели)

Не передаются: Telegram ID, имя пользователя, любые идентифицирующие данные.

Согласие на трансграничную передачу: Принимая настоящую Политику, Пользователь даёт согласие на трансграничную передачу своих персональных данных в компанию Anthropic (США) в целях формирования персонализированных рекомендаций по питанию. Передача осуществляется в минимально необходимом объёме — без идентифицирующих данных.

4.2. Обработка платежей осуществляется через ЮKassa (НКО «ЮМани», Россия). Данные банковских карт обрабатываются ЮKassa в соответствии с требованиями PCI DSS. Оператор не имеет доступа к данным карт.

4.3. Данные не передаются иным третьим лицам, за исключением случаев, предусмотренных законодательством РФ (по запросу уполномоченных органов).

5. Хранение данных

5.1. Персональные данные хранятся на серверах, расположенных на территории Российской Федерации.

5.2. Сроки хранения:

  • Данные анкеты и анализов: до 2 лет с момента последнего обращения или до удаления аккаунта Пользователем
  • История платежей: 3 года (требование налогового законодательства)
  • PDF-файлы: удаляются сразу после обработки (не хранятся)

5.3. По истечении сроков хранения данные удаляются безвозвратно.

6. Защита данных

6.1. Оператор принимает следующие меры для защиты персональных данных:

  • Шифрование соединений (HTTPS/TLS)
  • Хранение данных в защищённой базе данных с ограниченным доступом
  • Регулярное обновление программного обеспечения
  • Доступ к данным имеет только Оператор

6.2. Оператор обязуется уведомить Пользователя о любом инциденте утечки данных в кратчайшие сроки.

7. Права пользователя

7.1. Пользователь имеет право:

  • Узнать, какие данные хранятся: отправить команду /my_data в Боте
  • Удалить все свои данные: отправить команду /delete_data в Боте
  • Отозвать согласие на обработку: написать @oruzhiye74 или удалить данные через Бот
  • Получить копию данных: по запросу через @oruzhiye74

7.2. При удалении данных:

  • Удаляются: анкета, анализы, рекомендации, история меню, данные о БАД
  • Сохраняются: история платежей (на срок, установленный законодательством)

7.3. Удаление данных необратимо. Для повторного использования Бота потребуется заново пройти анкетирование и дать согласие.

8. Файлы cookie и аналитика

8.1. Бот работает исключительно в среде Telegram и не использует файлы cookie, пиксели отслеживания или иные технологии веб-аналитики.

9. Возрастные ограничения

9.1. Бот предназначен для лиц, достигших 18 лет. Оператор не осуществляет целенаправленный сбор данных несовершеннолетних.

10. Изменения в политике

10.1. Оператор вправе вносить изменения в настоящую Политику. Актуальная версия всегда доступна в Боте (раздел «Политика конфиденциальности»).

10.2. Продолжение использования Бота после изменения Политики означает согласие с новой редакцией.
